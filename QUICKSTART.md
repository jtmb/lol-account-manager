# Quick Start Guide

## First Time Setup (5 minutes)

### Step 1: Install Python
1. Go to https://www.python.org/downloads/
2. Click "Download Python 3.11" (or latest 3.9+)
3. Run the installer
4. ⚠️ **IMPORTANT**: Check the box "Add Python to PATH"
5. Click "Install Now"
6. Restart your computer

### Step 2: Download & Run
1. Click the green "Code" button on GitHub and select "Download ZIP"
2. Extract the ZIP file to a folder (e.g., `C:\Games\LoL-Manager`)
3. Double-click **`run.bat`** in the folder
4. Wait for dependencies to install (this takes 1-2 minutes)
5. The app will launch automatically!

## First Use

### 1️⃣ Set Master Password
- Choose a strong password (8+ characters recommended)
- Confirm it by typing again
- Click "OK"
- ⚠️ **Remember this!** If you forget it, you'll need to start over

### 2️⃣ Add Your Accounts
- Click **"+ Add Account"**
- Enter your League username (or email)
- Enter your password
- Give it a nickname (optional, e.g., "Main", "Smurf")
- Click "Add Account"
- Repeat for each account you want to save

### 3️⃣ Launch and Play!
- Select an account from the list
- Click **"Launch Selected Account"**
- Wait for the Riot Client to appear
- Log in if needed (credentials may auto-fill)
- League of Legends will launch automatically
- Enjoy!

## Troubleshooting

**"Python not found"**
- Python not installed or PATH not set
- Install from https://www.python.org/
- Check "Add Python to PATH" during install
- Restart your computer

**"League of Legends not found"**
- Make sure League of Legends is installed
- If installed in custom location, see DEVELOPER.md for setup

**Forgot Master Password?**
- Delete `%APPDATA%\LoLAccountManager\master.key`
- Restart the app
- Create a new master password
- Note: You'll lose your saved accounts
