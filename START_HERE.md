# 🎮 START HERE

## Step 1: Just Click! 👆

**Double-click `run.bat`**

That's it. Everything else is automatic.

---

## What Happens

```
You click run.bat
    ↓
It checks for Python
    ↓
If Python missing → Downloads & installs automatically
    ↓
Sets up the app automatically
    ↓
App launches! 🚀
```

**First time: 2-5 minutes**  
**After that: Instant!**

---

## If run.bat Doesn't Work

Try this instead:
- Right-click `run.ps1` 
- Select "Run with PowerShell"
- Click through any prompts

---

## First Use

1. Set a master password (write it down somewhere safe!)
2. Click "+ Add Account"
3. Enter your League username and password
4. Name it something like "Main" or "Smurf"
5. Click "Launch Selected Account"
6. Play!
